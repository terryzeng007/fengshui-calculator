# 风水计算器 - 苹果风格版

## 1. 项目概述
- **项目名称**: 八字命运馆
- **项目类型**: 静态网站 (单页应用)
- **核心功能**: 完整的八字命理分析，包含五行、八字、运势、事业、财运、感情等
- **目标用户**: 对中国传统命理感兴趣的用户

## 2. UI/UX 规范 (苹果风格)

### 布局结构
- **导航栏**: 固定顶部，毛玻璃效果
- **首页**: 品牌展示 + 输入区域
- **结果页**: 多卡片分区域展示
- **底部**: 简洁Tab导航

### 视觉设计 (苹果风格)

#### 色彩方案
- **主背景**: #000000 (纯黑)
- **卡片背景**: rgba(255,255,255,0.08)
- **强调色**: #007AFF (苹果蓝)
- **辅助色**: #30D158 (苹果绿)
- **警示色**: #FF453A (苹果红)
- **文字主色**: #FFFFFF
- **文字次色**: rgba(255,255,255,0.6)
- **边框色**: rgba(255,255,255,0.1)

#### 苹果风格特点
- 大圆角: 20px-30px
- 毛玻璃: backdrop-filter: blur(30px)
- 柔和阴影: 0 20px 40px rgba(0,0,0,0.3)
- 渐变高光: 顶部微渐变
- 流畅动画: 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94)

#### 字体
- **标题**: SF Pro Display (system-ui)
- **正文**: SF Pro Text (system-ui)
- **中文**: -apple-system, BlinkMacSystemFont, "PingFang SC"

#### 间距
- 容器最大宽度: 1200px
- 卡片内边距: 28px
- 区块间距: 24px
- 卡片圆角: 24px

### 组件

#### 输入组件
- 年月日时四字段时间选择器
- 性别开关 (iOS风格)
- 玻璃拟态输入框

#### 结果卡片 (8个)
1. **命主信息**: 五行属性、纳音、命主星
2. **八字详情**: 年月日时四柱
3. **性格分析**: 五行对应的性格特点
4. **事业运程**: 适合职业、工作方向
5. **财富运势**: 财运分析、投资建议
6. **感情分析**: 桃花运、适合对象
7. **幸运指引**: 颜色、数字、方位
8. **今日运势**: 每日运势提醒

## 3. 功能规范

### 核心功能

#### 1. 完整八字计算
- 年柱: 天干地支 + 五行
- 月柱: 天干地支 + 五行  
- 日柱: 天干地支 + 五行 (日主)
- 时柱: 天干地支 + 五行

#### 2. 五行分析
- 日主属性判断
- 五行强弱分析
- 喜用神判定
- 纳音五行

#### 3. 性格特点
- 根据五行属性分析性格
- 优点缺点解读
- 行为模式

#### 4. 事业分析
- 适合行业推荐
- 职业方向建议
- 贵人方位

#### 5. 财富运势
- 财运等级评估
- 理财建议
- 花钱性格

#### 6. 感情分析
- 桃花运强弱
- 理想对象类型
- 婚姻建议

#### 7. 幸运指引
- 幸运颜色 (3-5个)
- 幸运数字 (3-5个)
- 吉祥方位
- 适合数字

#### 8. 今日运势
- 综合运势
- 宜忌事项

### 天干地支计算
- 天干: 甲1 乙2 丙3 丁4 戊5 己6 庚7 辛8 壬9 癸0
- 地支: 子1 丑2 寅3 卯4 辰5 巳6 午7 未8 申9 酉10 戌11 亥0
- 日柱需计算公历转农历再算

### 五行对应
- 木: 青、绿
- 火: 红、紫、橙
- 土: 黄、棕、米
- 金: 白、金、银
- 水: 黑、蓝
