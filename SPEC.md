# 风水计算器 - 项目规范

## 1. 项目概述
- **项目名称**: 风水计算器 (Feng Shui Calculator)
- **项目类型**: 静态网站 (单页应用)
- **核心功能**: 基于出生年份计算五行属性、八字基础信息、幸运颜色和数字
- **目标用户**: 对中国传统风水和八字感兴趣的用户

## 2. UI/UX 规范

### 布局结构
- **Header**: 固定顶部，包含Logo和标题
- **Hero区域**: 中央大标题 + 副标题
- **输入区域**: 出生年份输入 + 性别选择
- **结果区域**: 显示计算结果卡片
- **Footer**: 简洁版权信息

#### 结果卡片 (4个)
1. **五行属性**: 显示主五行 + 纳音
2. **八字基础**: 年柱、月柱、日柱、时柱
3. **幸运颜色**: 3个推荐颜色 (色块展示)
4. **幸运数字**: 3个推荐数字

#### 按钮
- 计算按钮: 渐变背景 (#e94560 -> #ff6b6b)
- 悬停: 放大1.05 + 发光效果

## 3. 功能规范

### 核心功能
1. **五行计算**
   - 根据天干地支计算纳音五行
   - 年柱 = 天干 + 地支(年支)
   
2. **八字计算**
   - 年柱: 出生年份的天干地支
   - 月柱: 基于年干计算月干 + 当月地支
   - 日柱: 简化计算 (略)
   - 时柱: 简化计算 (略)

3. **幸运颜色**
   - 根据五行属性匹配
   - 木: 绿、青
   - 火: 红、紫
   - 土: 黄、棕
   - 金: 白、金
   - 水: 黑、蓝

4. **幸运数字**
   - 根据五行生克关系推荐

### 计算逻辑
- 天干: 甲乙丙丁戊己庚辛壬癸 (对应0-9)
- 地支: 子丑寅卯辰巳午未申酉戌亥 (对应0-11)
- 纳音五行口诀应用
